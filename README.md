Sistema de gestión de países

Descripción del programa:
Programa desarrollado en Python que permite gestionar información de distintos países almacenados en un archivo CSV.
El sistema permite agregar, actualizar, buscar, filtrar, ordenar y obtener estadísticas sobre los países registrados mediante un menú interactivo y persistente.

Instrucciones de uso:
1. Abrir el proyecto en Visual Studio Code.
2. Verificar que los archivos main.py y paises.csv se encuentren en la misma carpeta.
3. Ejecutar el archivo 'main.py'.
4. Interactuar con las distintas opciones del menú para explorarlo.

Ejemplos de uso:
Entrada / Salida
- Agregar un nuevo país (se ingresan todos los datos) / El país se guarda en el archivo CSV.
- Buscar un país por nombre / Con búsqueda parcial o exacta, se muestran todos los datos del país encontrado.
- Filtrar por continente / Se muestran los países pertenecientes al continente seleccionado.
- Mostrar estadísticas / Según lo que se solicite, se muestran los resultados calculados.
